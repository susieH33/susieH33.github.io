Tutorial: Basic Tables

Notes: use the table tag to make a table and the td tag to make a cell in a table, it stands for table data. For example:
Susie Sadye Meghan

To define different rows use the tr tag which stands for table row. For now, there should be the same number of rows and columns/cells
Susie Sadye Meghan
Jun 6 Oct 28 Nov 2

You can make table headers using the th tag.
Person 1 Person 2 Person 3
Susie Sadye Meghan
Jun 6 Oct 28 Nov 2
