JavaScript Statements Tutorial

A computer program is a list of instructions to be executed by a computer. These instructions are called statements.

JavaScript statements are composed of: Values, Operators, Expressions, Keyword, and Comments. Statements are executed one by one in the same order that they are written

Semicolons

Semicolons separate JS statements. Add a semicolon at the end of each executable statement. Multiple statements on one line are allowed when separated by a semicolon.

White Space

JavaScript ignores white spaces but it's recommended to use them to make script more readable

Syntax

JavaScript has variables and literals (fixed values). Variables can be defined using 'var' or 'let'. It's very similar to java.

Comments

Use // to define comments in JavaScript
