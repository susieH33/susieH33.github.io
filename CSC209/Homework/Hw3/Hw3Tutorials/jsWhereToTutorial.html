JavaScript Where To Tutorial

In HTML, js code can be inserted between 'script' tags.

JS Functions and Events

In JS a function is a block of code that can be executed when called. Ex a function can be called when an event occurs such as clicking a button.

JS scripts can be placed in the body, head, or both of an HTML page

External JavaScript

Scripts can also be placed in external files with .js file extensions. This is practice when the same code is used in numerous web pages. Put the name of the script file in the src attribute of a script tag.

Demo JS using an external JS file

External scripts have advantages including separating HTML and code, making files easier to read, and using several scripts in one page

External References

External scripts can be referenced with a full URL, with a file path, or without a path
