JavaScript Data Types Tutorial

JS has 8 data types

  • String
  • Number
  • Bigint
  • Boolean
  • Undefined
  • Null
  • Symbol

Object Data Type

The object data type can contain both built in obkects and user defined objects. Examples of built in objects include: arrays, maps, dates, sets, intarrays, floatarrays, promises, and more.
