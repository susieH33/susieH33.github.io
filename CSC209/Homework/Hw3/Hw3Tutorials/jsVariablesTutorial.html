JavaScript Variables Tutorial

Variables can be declared in 4 ways: automatically, using 'var', using 'let', and using 'const'. It's considered best practice to always declare variables before use and declare them at the beginning of a script.

When to use var, let, or const?

Use 'const' if the value and type should not change. Only use 'let' if you can't use 'const'. Only use 'var' if you haveeeee to support old browsers.

Data types and values will be undefined until you declare them using =. If you redeclare a variable with 'var' it will not lose its value but you cannot redeclare variables using 'let' or 'const'.
