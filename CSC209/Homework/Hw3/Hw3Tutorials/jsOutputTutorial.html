JavaScript Output Tutorial

JavaScript can display data in different ways: to an HTML element, to HTML output, to an altert box, and to the browser console.

innerHTML

Use the 'document.getElementById(id)' method followed by .innerHTML. This will change the content of the element.

document.write()

Never call document.write after the document has finished loading (ex in an onclick of a button). It will overwrite the whole document. This function should only be used for testing.

window.alert()

You can output text into an alert box. Note the 'window' keyword is optional

console.log()

You can use console.log() to display data for debugging purposes

window.print()

JS doesn't have a print object or method. You can use window.print() ot print the content of the current browser.
