Let, Const, Operators, Arithmetic, and Assignment Tutorials

JavaScript Let

The 'let' keyword was introduced in 2015. Variables declared with 'let' have Block Scope, must be declared before use, and cannot be redeclared in the same scope.

Block Scope vs Global Scope

Variables declared with 'let' and 'const' have block scope and cannot be accessed outside the {} block they were declared in.

Variables declared with 'var' always have global scope meaning they can be accessed from outside the {} block.

You should declare variables with 'const' when you know the variable won't be changed such as new Arrays, Objects, Functions, or RegExp

This does not mean the value of the variable has to be constant, it means it's a constant reference to a value. Thus, you cannot reassign a const value/array/object but you can change the elements of a constant array or properties of a constant object.
