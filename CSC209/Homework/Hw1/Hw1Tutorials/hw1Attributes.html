Homework 1 Attributes Tutorial

HTML elements can have attributes that provide more information about the elements. Attributes are specified in the start tag and come in name/value pairs. Ex name = "value"

Common HTML Attributes

href Attribute
The 'a' tag defines a hyperlink and the href attribute contains the URL of the page the link goes to. Example: Visit W3Schools

scr Attribute
The 'img' tag is used to put an image into an HTML page and specifies the path to the image. Example:
There are two URL types that can be used in the src attribute: Absolute URL and Relative URL. Aboslute URLs are links to images hosted within another website and start with "https://...." whereas relative URLs link to an image within the website and can be relative to the current page (no slash to start, ex "pic.png) or relative to the domain (slash at front, ex "/images/pic.png"). It's best to use relative URLs since they won't break if the domain changes.

lang Attribute
The lang attribute is inside the html tag at the top of an HTML file and is used to specify the language of the web page. To declare the language as english: lang="en"
Country codes can also be added: lang="en-US"

Other attribute notes
HTML standard doesn't require lowercase attribute names but it's recommended to use lowercase attributes. It's also recommended to use quotes (single or double) around attribute values to ensure the entire string is displayed.
