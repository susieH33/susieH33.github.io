Tutorial: HTML Classes

The 'class' attribute can be used to point to class names in style sheets and can be used by JavaScript to access elements with that class name. Multiple HTML elements can share the same class. Classes can be used on any html element and are case sensitive

The class content goes in the style element at the top. Below are three examples that all use the class defined in this file. You can see that all the div elements have the same styling because they use the same class.

London is the capital of England.
