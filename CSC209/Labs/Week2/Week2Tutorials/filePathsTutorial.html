File Paths Tutorial

File paths describe the location of a file in a website's folder structure. Examples:

"picture.jpg" "images/picture.jpg" "/images/picture.jpg" "../picture.jpg"
The "picture.jpg" file is located in the same folder as the current page The "picture.jpg" file is located in the images folder in the current folder The "picture.jpg" file is located in the images folder at the root of the current web The "picture.jpg" file is located in the folder one level up from the current folder

Absolute File Paths

Absolute file paths use the full URL to a file, ex https://www.w3schools.com/images/picture.jpg

Relative File Paths

Relative file paths point to files relative to current pages. This is best practice and will allow links to work on your computer and other domains
