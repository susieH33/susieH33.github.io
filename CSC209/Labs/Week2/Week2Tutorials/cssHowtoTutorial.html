External CSS using syle sheets

You can use an external style sheet to change the look of an entire website by changing just one file. Include a reference to the sheet file inside the 'link' element in the head section.

Internal CSS

the header of this section was set using internal CSS that is defined in the 'style' element in the head section.

Inline CSS

Inline CSS can be used to apply a unique style for a single element using the syle attribute in the start tag.
