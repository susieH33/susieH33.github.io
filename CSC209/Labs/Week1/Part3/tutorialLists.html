  • Strawberry

You can also change the shape of the list marker to disc, circle, square, or none by using style="list-style-type:[type];" in the first tag. For example:
  • Strawberry
  • Apple
  • Mango
Unordered lists can also be nested:
  • Strawberry
  • Apple
    • Honeycrisp
    • Gala
  • Mango

Ordered Lists
  1. First
  2. Second
  3. Third
Using the "type" attribute, you can change the item list marker to be numbers ("1"), uppercase letters ("A"), lowercase letters ("a"), uppercase roman numerals ("I"), or lowercase roman numerals ("i"). Change the type attribute in the first tag.
  1. First
  2. Second
  3. Third
Use the "start" attribute to change the starting count number
  1. Tenth
  2. Eleventh
  3. Twelfth
Like unordered lists, you can also nest ordered lists.
